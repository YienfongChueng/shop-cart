# shop-cart

## 主要功能

### 1.购物车列表页
```
产品数量的增加和减少
产品的金额根据产品数量动态计算
总的结算金额能够根据选中的产品动态计算
全选和反选
删除产品弹窗提示
```
### 2.地址列表页
```
地址项默认展示3项，提供展开更多和收起
删除地址
设置默认地址
```
